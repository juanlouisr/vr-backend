# Railway deployment

Create Empty Project, add mysql, add service from GitHub repo (vr-backend)

setup vr-backend service variable from created MySql project

```env
spring_profiles_active=prod
PROD_DB_HOST=
PROD_DB_PORT=
PROD_DB_NAME=railway
PROD_DB_USERNAME=root
PROD_DB_PASSWORD=
PORT=8080
```
